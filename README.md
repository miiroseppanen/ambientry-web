# Sayry-web

Minimalistinen skeleton ~ Suomen Ambientyhdistyksen sivulle.

## Sisällön hallinta
- `content.json` listaa osiot ja niihin liittyvät markdown-tiedostot.
- Jokainen markdown-tiedosto luo oman osionsa sivulle.

## Käyttö
Avaa `index.html` selaimessa tai tarjoa tiedostot kevyellä HTTP-palvelimella.
